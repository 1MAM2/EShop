# üõí EShop

A **full-stack e-commerce application** built with React, TypeScript, and ASP.NET Core.  
This project provides a basic e-commerce experience where users can browse products, add items to the cart, and experience the shopping process.

---

## üõ† Technologies Used

| Layer        | Technology                    |
|--------------|-------------------------------|
| Frontend     | React, TypeScript, Context API |
| Backend      | ASP.NET Core Web API           |
| Database     | Entity Framework Core + MySQL |
| UI           | Responsive Design (CSS, Flexbox, Grid) |
| Auth         | JWT Authentication (planned)  |
